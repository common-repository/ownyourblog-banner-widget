== Description ==

Simple, but very useful and powerful plaugin, which adds to your Wordpress-powered website a special widget which can show unlimited banners in any siderbar of you theme.

All you need to do is to place it in the right place, add title, URL of the banner's image and your affiliate link.

**Features:**

* Unlimited banners to any sidebar of your website;
* Show up any .jpg, animated .gif or transparent .png image;
* Customize the width of every banner with 'width' parameter. No need to fill in the height.

**Ownyourblog Banner Widget is bulletproof solution tested on many blogs.**


== Installation ==

To set the plugin up, you need to:

1. Choose 'Plugins - Add new', then 'Upload'. Select ownyourblog-banner-widget.zip file and upload it.
2. Activate the plugin through the 'Plugins' menu in WordPress

That's all!
